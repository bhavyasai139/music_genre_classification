# Music Genre Classification

## Overview

This project classifies music into different genres using machine learning techniques. It analyzes audio features and predicts the genre of a given music file.

## Features

* Music genre prediction
* Deep learning-based classification
* Web-based user interface
* Audio feature extraction

## Tech Stack

* Python
* TensorFlow / Keras
* NumPy, Pandas
* HTML, CSS, JavaScript

## Project Files

* task22.py – Backend logic
* index.html – Frontend
* styles.css – Styling
* tasktemp.js – JavaScript functionality
* final_model_completed.ipynb – Model training and experimentation

## How to Run

1. Install required libraries
2. Run the backend:

   ```
   python task22.py
   ```
3. Open `index.html` in a browser
